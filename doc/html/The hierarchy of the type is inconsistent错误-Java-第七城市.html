The hierarchy of the type is inconsistent错误

2013-01-19 19:27:29cnblogs.com-yanzhenxing-点击数:

 

编写java程序时，出现The hierarchy of the type is inconsistent 错误。

 

可能的原因：自己的类继承于某个类，这个类或者这个类继承的类或者再往上继承的某个类所在的jar包没有被引入。
