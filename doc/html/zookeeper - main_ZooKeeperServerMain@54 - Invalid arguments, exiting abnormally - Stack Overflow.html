There may be some changes in the Zookeeper 3.4.6. For this version start the server as shown below:

bin\zkServer.cmd

E.g.

D:\zoo\zookeeper-3.4.6>bin\zkServer.cmd

Note, you don't need to pass the command line arg "start"
